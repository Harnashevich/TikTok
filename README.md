# TikTok

TikTik app on iOS with the ability to record, watch and save video.

Technology stack:

• UIKit (programmatically)<br />
• MVVM architecture<br />
• Firebase storage and authentication<br />
• UICollectionViewCompositionalLayout<br />
• AVFoundation<br />
• URLSession<br />
